# /moon

Show the current moon phase and receive some moon wisdom. Use this command for a quick, mystical check-in with lunar energy—perfect for daily reflection, planning rituals, or just a dose of cosmic inspiration.

<hr className="md-divider-gradient" />

<h3 className="heading-with-icon">
  <img src="/img/icons/chat.svg" alt="" className="heading-icon" />
  <span>Usage</span>
</h3>
```bash
/moon
```
:::secondary
No options or inputs are required—just run the command to receive the current phase and guidance.
:::

<h3 className="heading-with-icon">

</h3>
```bash
/moon
```

- Display the current moon phase (e.g., Waxing Crescent, Full Moon, etc.)
- Show a brief piece of “moon wisdom” or a supportive reflection tied to the lunar phase
- Offer a gentle nudge or insight for your day, week, or magical practice
