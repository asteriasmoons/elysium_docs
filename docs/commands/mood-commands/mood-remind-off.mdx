# /mood remind off 

Turn off your mood reminder notifications. Use this command if you no longer want to receive scheduled reminders to log your mood.

<hr className="md-divider-gradient" />

<h3 className="heading-with-icon">
  <img src="/img/icons/chat.svg" alt="" className="heading-icon" />
  <span>Usage</span>
</h3>
```bash
/mood remind off
```

:::secondary
**No options or inputs are required—just run the command to disable mood reminders.**
:::

<hr className="md-divider-gradient" />

<h3 className="heading-with-icon">
  <img src="/img/icons/example.svg" alt="" className="heading-icon" />
  <span>Example</span>
</h3>
```bash
/mood remind off
```
This will:
- Immediately disable all scheduled mood log reminders for your account
- Stop Elysium from sending you prompts to log your mood at set times

<hr className="md-divider-gradient" />

<h3 className="heading-with-icon">
  <img src="/img/icons/notes.svg" alt="" className="heading-icon" />
  <span>Notes</span>
</h3>
> - Only affects your personal reminder—other users’ reminders are unaffected
> - You can turn reminders back on any time with `/mood remind set`
> - Turning off reminders does not delete your past mood logs or stats
> - Useful if you want to take a break or prefer to log moods manually

<hr className="md-divider-gradient" />

<h3 className="heading-with-icon">
  <img src="/img/icons/related.svg" alt="" className="heading-icon" />
  <span>Related Commands</span>
</h3>
`/mood remind set` – Set up a mood log reminder

`/mood remind status` – Check if your reminder is currently on

`/mood log` – Log your mood manually

`/mood stats` – View your mood statistics
