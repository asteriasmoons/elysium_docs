# /mood remind set

Set up a personal reminder to log your mood at a specific time and frequency. Use this command to have Elysium send you scheduled notifications to check in on your mood—perfect for building self-awareness and consistent emotional tracking.

<hr className="md-divider-gradient" />

<h3 className="heading-with-icon">
  <img src="/img/icons/chat.svg" alt="" className="heading-icon" />
  <span>Usage</span>
</h3>
```bash
/mood remind set <hour> <minute> <frequency> [timezone]
```
- `<hour>` – The hour you want the reminder to be sent (24-hour format, e.g., 20 for 8pm)
- `<minute>` – The minute you want the reminder to be sent (e.g., 30 for :30 after the hour)
- `<frequency>` – How often you want to be reminded (e.g., daily, weekly)
- `[timezone]` – (Optional) Your timezone in Region/City format (e.g., America/Chicago). If omitted, defaults to the server default.

:::secondary
**Remember that time in the bot runs on 24 hour format. To figure out what time it is at per se 5PM, just add 12 to it to get 17**
:::

<hr className="md-divider-gradient" />

<h3 className="heading-with-icon">
  <img src="/img/icons/example.svg" alt="" className="heading-icon" />
  <span>Example</span>
</h3>
```bash
/mood remind set 20 30 daily America/Chicago
```
This will:
- Schedule a daily mood reminder at 8:30pm in the America/Chicago timezone
- Elysium will send you a prompt at the set time, reminding you to log your mood
- You’ll receive reminders according to your specified frequency and timezone

<hr className="md-divider-gradient" />

<h3 className="heading-with-icon">
  <img src="/img/icons/notes.svg" alt="" className="heading-icon" />
  <span>Notes</span>
</h3>
> - Time uses a 24-hour clock; be sure to enter your desired hour and minute correctly
> - You can change your reminder time or frequency anytime by running this command again
> - If you don’t set a timezone, the bot uses the server default
> - Reminders are private and only sent to you—not the entire server
