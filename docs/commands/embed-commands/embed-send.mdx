# /embed send

Post one of your saved embeds directly into the current channel. Use this to publicly share an embed you've created and stored with Pandoryx.

<hr className="md-divider-gradient" />

<h3 className="heading-with-icon">
  <img src="/img/icons/chat.svg" alt="" className="heading-icon" />
  <span>Usage</span>
</h3>
```bash
/embed send <name>
```
- `<name>` – The name of the embed you want to send. Must match one that has already been created and saved.

:::success
This command sends the embed to the channel you run it in, so be sure you’re in the right place before using it!
:::

<hr className="md-divider-gradient" />

<h3 className="heading-with-icon">
  <img src="/img/icons/example.svg" alt="" className="heading-icon" />
  <span>Example</span>
</h3>
```bash
/embed send rules
```

<img
  src="/img/screenshots/embedsend.png"
  alt="Description"
  className="doc-image"
/>

This will:
- Send the embed named rules into the current text channel
- Display it exactly as it was configured using `/embed create` or `/embed edit`

<hr className="md-divider-gradient" />

<h3 className="heading-with-icon">
  <img src="/img/icons/notes.svg" alt="" className="heading-icon" />
  <span>Notes</span>
</h3>
> - You can send the same embed multiple times or in multiple channels.
> - If you need to preview the embed without sending it, use `/embed view` instead.
