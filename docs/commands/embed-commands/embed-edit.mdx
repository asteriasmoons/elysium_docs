# /embed edit

Edit a saved embed using an interactive editor with buttons and modals. This lets you update any part of your embed without starting over.

<hr className="md-divider-gradient" />

<h3 className="heading-with-icon">
  <img src="/img/icons/chat.svg" alt="" className="heading-icon" />
  <span>Usage</span>
</h3>
```bash
/embed edit <name>
```
- `<name>` – The name of the embed you want to edit. Must match a previously created embed.

:::success
The editor uses the same button-based interface as `/embed create`, allowing you to quickly adjust the title, description, colors, images, and more.
:::

<hr className="md-divider-gradient" />

<h3 className="heading-with-icon">
  <img src="/img/icons/example.svg" alt="" className="heading-icon" />
  <span>Example</span>
</h3>
```bash
/embed edit rules
```

<h3 className="heading-with-icon">
  <img src="/img/icons/notes.svg" alt="" className="heading-icon" />
  <span>Notes</span>
</h3>
> - Changes are saved instantly — there’s no need to re-run `/embed create.`
> - If you want to preview the updated embed before sending it, use `/embed view.`
> - You can repost the edited embed anytime using `/embed send.`
