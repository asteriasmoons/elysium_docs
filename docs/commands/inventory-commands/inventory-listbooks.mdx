# /inventory listbooks 

View Elysium’s global inventory of books. Use this command to browse all the titles that have been added by the community, complete with author information and Goodreads links for easy reference and discovery.

<hr className="md-divider-gradient" />

<h3 className="heading-with-icon">
  <img src="/img/icons/chat.svg" alt="" className="heading-icon" />
  <span>Usage</span>
</h3>
```bash
/inventory listbooks
```

:::warning
**No options or inputs are required—just run the command to view the inventory.**
:::

<hr className="md-divider-gradient" />

<h3 className="heading-with-icon">
  <img src="/img/icons/example.svg" alt="" className="heading-icon" />
  <span>Example</span>
</h3>
```bash
/inventory listbooks
```
This will:
- Display an embed or paginated list of all books currently in Elysium’s global inventory
- Show each book’s title, author, and a clickable Goodreads link
- Help you discover new reads, check for duplicates, or recommend books to others

<hr className="md-divider-gradient" />

<h3 className="heading-with-icon">
  <img src="/img/icons/notes.svg" alt="" className="heading-icon" />
  <span>Notes</span>
</h3>
> - The inventory is global—everyone using Elysium can view and contribute to the list
> - The list may be paginated if there are many entries; use the provided navigation to scroll through
> - For best results, always use accurate titles and author names when adding books
> - You can remove a book (server owner only) or add new ones at any time

<hr className="md-divider-gradient" />

<h3 className="heading-with-icon">
  <img src="/img/icons/related.svg" alt="" className="heading-icon" />
  <span>Related Commands</span>
</h3>
`/inventory addbook` – Add a new book to the inventory

`/inventory bookremove` – Remove a book from the global inventory (server owner only)
